# Book Tracker — Трекер прочитанных книг

**Автор:** Сергей Усольцев
**Проект:** Book Tracker  
**Дата создания:** апрель 2026

## Описание программы

**Book Tracker** — это GUI‑приложение для учёта прочитанных книг. Программа позволяет:

* добавлять книги с указанием названия, автора, жанра и количества страниц;
* просматривать список всех книг в табличном виде;
* фильтровать книги по жанру и количеству страниц;
* сохранять данные в JSON‑файл;
* загружать данные из JSON‑файла;
* удалять отдельные записи или очищать весь список;
* проверять корректность ввода данных.


Приложение разработано на Python с использованием библиотеки Tkinter для графического интерфейса.

## Требования для запуска

Для работы программы необходимо:

* Python 3.8 или выше;
* стандартные библиотеки Python (`tkinter`, `json`, `os`) — не требуют отдельной установки.

## Как пользоваться
Добавление книги
Заполните все поля формы:

1. Название книги — введите название;

- Автор — укажите автора;

- Жанр — укажите жанр;

- Количество страниц — введите число.

2. Нажмите кнопку «Добавить книгу».

 Книга появится в таблице ниже.

## Фильтрация книг
1. Выберите параметры фильтрации:

- в выпадающем списке «Жанр» выберите нужный жанр или «Все»;

- в выпадающем списке «Страниц >» выберите порог или «Все».

2. Нажмите «Применить фильтр».

- Таблица обновится, отображая только книги, соответствующие фильтрам.

- Для сброса фильтров нажмите «Сбросить».

## Сохранение и загрузка данных
«Сохранить в JSON» — сохраняет текущий список книг в файл books_data.json.

«Загрузить из JSON» — загружает книги из файла books_data.json в таблицу.

## Удаление книг
«Удалить выбранную» — удаляет книгу, выделенную в таблице.

«Очистить всё» — удаляет все книги из списка и файла данных.

## Проверка корректности ввода
Программа проверяет:

Заполнение всех полей — если какое‑то поле пустое, появится сообщение об ошибке.

Корректность количества страниц — значение должно быть положительным целым числом. В противном случае появится сообщение: «Количество страниц должно быть числом!» или «Количество страниц должно быть положительным числом!».
